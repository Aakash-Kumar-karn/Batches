* as soon as res.end line is exceuted there is not further 
exceution of lower handler function
even if they satisifies all the condition.
* when a request is send then it will execute all the route 
handlers serially 
* next sends request  to the handler below you 

* **if** request hit's your handler
        use -> it will always execute the function 
        get -> route + get method  then it will execute
        post -> route + get method  then it will execute
        patch -> route + get method  then it will execute
        delete -> route + get method  then it will execute
